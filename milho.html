Pesquisa simulada

Simulação de Pesquisa com Inteligência Artificial

Este espaço simula o uso de uma Inteligência Artificial para apoiar pesquisas sobre agricultura. As respostas não são geradas por uma IA real em tempo real. Elas são carregadas de um arquivo JSON criado para fins educacionais.

Esta é uma simulação. O sistema pesquisa respostas prontas em um arquivo JSON local.

Nota sobre a simulação

Para manter o projeto dentro da proposta de programação front-end, esta área não utiliza API real de Inteligência Artificial, não envia dados para serviços externos e não depende de chave de acesso.

A consulta é feita no arquivo ia_simulada.json, localizado na pasta dados. Esse arquivo contém respostas previamente organizadas sobre agricultura familiar, sustentabilidade, riscos climáticos, mercado e culturas agrícolas.

O conteúdo textual do arquivo de simulação foi produzido com apoio do ChatGPT e deve ser entendido como material educativo complementar. As informações importantes devem ser conferidas nas fontes oficiais indicadas no projeto.
