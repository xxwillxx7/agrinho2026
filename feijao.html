Nota sobre o uso da API

O recurso de Inteligência Artificial utilizado neste projeto funciona por meio de uma API acessada através de um arquivo fornecido pelo professor orientador. Esse arquivo faz a comunicação entre o site e o serviço de IA, evitando que chaves sensíveis sejam expostas diretamente no código HTML, CSS ou JavaScript do projeto.

A IA é utilizada apenas como apoio para consultas relacionadas à agricultura. As respostas geradas devem ser conferidas com as fontes oficiais utilizadas na pesquisa do projeto.
