Sistema de Sugestões Agrícolas

Este projeto apresenta informações sobre culturas agrícolas, agricultura familiar, mercado, riscos climáticos e sustentabilidade, usando HTML, CSS, JavaScript e arquivos JSON.

O objetivo é mostrar como a tecnologia pode ajudar na organização e apresentação de dados sobre o campo, fortalecendo a relação entre produção agrícola, meio ambiente e sociedade.

Campo, tecnologia e sustentabilidade

Cada cultura agrícola aqui apresentada, possui uma página própria com dados organizados, imagem ilustrativa, fontes consultadas e apoio de leitura por JavaScript.

Como o projeto foi elaborado

  1. Análise do tema proposto pelo projeto.
  2. Pesquisa sobre culturas agrícolas relevantes no Paraná.
  3. Coleta de fontes confiáveis sobre essas culturas.
  4. Transformação das informações coletadas em arquivos JSON resumidos.
  5. Elaboração do site do projeto com HTML, CSS e JavaScript.

Sobre o projeto

Como o sistema funciona?

O site foi organizado para apresentar informações agrícolas de maneira simples, visual e interativa.

Arquivos JSON

As informações de cada cultura ficam armazenadas em arquivos separados dentro da pasta dados. O JavaScript lê esses arquivos e exibe o conteúdo na página.

JavaScript

O arquivo script.js é responsável por buscar os dados, montar o conteúdo na tela e organizar as informações de forma dinâmica.

Sustentabilidade

O projeto relaciona produção agrícola, agricultura familiar, riscos climáticos, mercado e equilíbrio ambiental.

Pesquisa e fontes

Fontes dos dados

As páginas das culturas apresentam as fontes utilizadas nos arquivos JSON. Entre as referências estão relatórios agrícolas, censos agropecuários e estudos técnicos.

Fontes utilizadas na pesquisa

  • Secretaria da Agricultura e do Abastecimento do Paraná - SEAB.
  • Departamento de Economia Rural do Paraná - DERAL.
  • Página oficial “Situação das Principais Culturas” - SEAB/DERAL.
  • Levantamento da Produção Agropecuária do Paraná - DERAL.
  • Relatórios e documentos técnicos sobre milho, soja e feijão no Paraná.
  • IBGE - Censo Agropecuário.
  • IPEA - Estudos sobre agricultura familiar brasileira.
  • UTFPR - Estudo sobre sistema de controle automático de termonebulizador agrícola para controle de geadas em plantações.
  • eduCAPES - Registro de trabalho técnico relacionado ao controle de geadas em plantações.
  • Pesquisa sobre a dinâmica da agricultura familiar em Toledo/PR com base nos Censos Agropecuários de 2006 e 2017.
  • Panorama da Agricultura Familiar e da Produção Orgânica no Paraná.
  • Materiais técnicos sobre agricultura familiar, produção orgânica, sustentabilidade e riscos climáticos.

Uso de Inteligência Artificial

  • Gemini: utilizado como apoio na criação das imagens ilustrativas das culturas agrícolas presentes no site.
  • ChatGPT: utilizado como apoio na revisão e melhoria dos códigos HTML, CSS e JavaScript.
  • ChatGPT: utilizado também para produzir o conteúdo textual do arquivo ia_simulada.json, criado para a simulação educativa de pesquisa com IA.
  • A área de pesquisa com IA presente nas páginas das culturas é uma simulação. Ela não utiliza API real, não envia dados para serviços externos e não depende de chave de acesso.
  • As respostas da simulação são carregadas de um arquivo JSON local, localizado na pasta dados, com o nome ia_simulada.json.
  • A simulação foi criada para demonstrar, de forma educativa, como uma IA poderia apoiar consultas sobre agricultura familiar, sustentabilidade, clima, mercado, feijão, milho e soja.
  • A Inteligência Artificial foi utilizada como ferramenta de apoio educacional, sem substituir a pesquisa, a organização dos dados e a autoria.
  • As informações apresentadas no projeto foram conferidas e organizadas a partir das fontes de pesquisa indicadas nas páginas das culturas.
